# Shadow Language Lab

[English](./README.md)

Shadow Language Lab 是一个本地优先的 shadowing 练习工具，可以把 SRT 字幕和音视频文件转成聚焦的口语跟读练习。

## 这是什么

Shadow Language Lab 是一个纯前端的英语 shadowing 练习工具。你可以导入视频/音频和 SRT 字幕，应用会把零散字幕自动组合成更完整的练习段，让你围绕一小段完整意思反复听、跟读、录音和评分。

它适合用来练习演讲、访谈、课程、播客、会议录音等素材的口语表达。

## 核心功能

- 一个按钮导入字幕和音视频文件，可以同时选，也可以分开选。
- 自动把 SRT 中过碎的字幕合并成几句话一组的练习段。
- 按时间码循环播放对应视频/音频片段。
- 没有媒体文件时，可使用浏览器 TTS 朗读字幕。
- 支持麦克风录音和浏览器语音识别。
- 根据识别文本计算准确度、语速和综合分。
- 支持手动自评分。
- 练习进度保存在本地浏览器中，刷新后可恢复。
- 媒体文件会尽量缓存到 IndexedDB，方便刷新后继续练。
- 设置面板可调整重复次数、播放速度和前后留白。

## 快速开始

这个项目不需要构建，也不需要后端服务。

```bash
python3 -m http.server 8766
```

然后打开：

```text
http://127.0.0.1:8766/
```

如果你在项目外层目录运行，请先进入项目目录：

```bash
cd shadow-language-lab
python3 -m http.server 8766
```

## 使用流程

1. 点击右上角“导入素材”。
2. 选择一个 `.srt` 字幕文件，也可以同时选择视频或音频文件。
3. 在左侧选择要练习的片段。
4. 点击播放按钮，循环听这一段。
5. 点击录音按钮进行跟读练习。
6. 查看识别结果、自动分数，或手动给自己评分。

## 数据与隐私

Shadow Language Lab 是 local-first 工具。字幕、媒体和练习记录默认都保存在你的浏览器本地，不会上传到服务器。

需要注意：浏览器不能在刷新后自动重新读取你磁盘上的原始视频路径，所以应用会尝试把你选择的媒体 Blob 缓存在 IndexedDB 中。如果文件太大、超过浏览器存储限制，刷新后需要重新导入媒体文件。

## 浏览器支持

推荐使用最新版 Chrome。语音识别、TTS、媒体缓存等能力依赖浏览器支持；如果某个能力不可用，应用仍可以作为字幕分段和循环播放工具使用。
